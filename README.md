# Flash Cards App
### HTML, CSS, JavaScript, Node, Express, Pug. 

## How to use:

Clone this repository:
  ```
  git clone https://github.com/sherialeliasjawed/flashcards.git
  ````

Install dependencies:
```
npm i 
```

In /flashcards/, use node or nodemon:

  Node:
```
node app.js
```
 Nodemon:
 ```
 nodemon
  ```
